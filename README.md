## The Problem

Indian agriculture faces critical challenges:
- **150 million farmers** lack access to expert advice
- **20-40% crop losses** due to undiagnosed plant diseases
- **Language barriers** — most agritech apps are English-only
- **Water scarcity** — inefficient irrigation wastes 40% of water
- **Market exploitation** by middlemen due to lack of price transparency
- **Equipment costs** — small farmers can't afford modern machinery
- **Mental health crisis** — farmer suicides are a national concern

## Our Solution

**Kissan Mitra** is a comprehensive AI-powered platform that addresses ALL these challenges through **13 integrated features**, accessible in **11 Indian languages**, with **voice I/O** and a **Material Design 3** interface.

## Tech Stack

### Frontend
- **React 19** + **TypeScript 5.9** + **Vite 7**
- **Tailwind CSS v4** with Material Design 3 / Google Stitch design system
- **react-leaflet** for interactive maps with OpenStreetMap
- **qrcode.react** for UPI QR code generation
- **i18next** + **react-i18next** for 11-language internationalization
- **Chart.js** for market price data visualization
- **marked** for Markdown rendering in chat

### Backend
- **FastAPI** (Python 3.11+) with Uvicorn ASGI server
- **LangGraph** + **LangChain** for multi-agent AI orchestration
- **ChromaDB** for RAG vector storage (mental health documents)
- **Groq API** (Llama 3.3 70B) for fast LLM inference
- **Google Gemini 2.5 Flash** for vision analysis + schemes chatbot
- **scikit-learn** for ML models (Random Forest, DBSCAN clustering)
- **TensorFlow/Keras** for Xception CNN disease classification (38 classes)
- **Tavily API** for real-time web search integration

### External APIs
| API | Purpose |
|-----|---------|
| OpenWeatherMap | Real-time weather + 5-day forecasts |
| Data.gov.in | Government commodity market prices |
| Tavily AI | Web search with source citations |
| Groq Cloud | Fast LLM inference (Llama 3.3 70B) |
| Google Gemini | Vision AI + conversational AI |

## Quick Start

### Prerequisites
- Python 3.11+
- Node.js 18+
- npm

### Installation

```bash
# Clone the repository
git clone https://github.com/Omkarop0808/kissanMitra.git
cd kissanMitra

# Backend setup
python -m venv venv
venv\Scripts\activate        # Windows
# source venv/bin/activate   # Linux/Mac
pip install -r requirements.txt

# Frontend setup
cd frontend
npm install
npm run build
cd ..

# Configure environment
cp .env.example .env
# Edit .env and add your API keys (see table below)

# Run the application
python app.py
```

Open `localhost:3000` in your browser.

### API Keys Required

| Service | Purpose | Free Tier | Get Key |
|---------|---------|-----------|---------|
| Groq | AI Assistant LLM | Yes | [console.groq.com](https://console.groq.com/keys) |
| Google Gemini | Vision + Schemes | Yes | [aistudio.google.com](https://aistudio.google.com/apikey) |
| OpenWeatherMap | Weather data | Yes | [openweathermap.org](https://openweathermap.org/api) |
| Data.gov.in | Market prices | Yes | [data.gov.in](https://data.gov.in) |
| Tavily | Web search (optional) | Yes | [app.tavily.com](https://app.tavily.com) |

All API keys have generous free tiers — no credit card required.

## Architecture

```
kissanMitra/
├── app.py                          # FastAPI main server (all API endpoints)
├── FarmerAssistant.py              # LangGraph multi-agent system (7 agents)
├── prediction.py                   # Xception CNN + Gemini Vision disease model
├── hotspot.py                      # DBSCAN clustering + Folium hotspot maps
├── weather_info.py                 # OpenWeatherMap API integration
├── requirements.txt                # Python dependencies
├── .env.example                    # Environment variables template
├── frontend/
│   ├── src/
│   │   ├── pages/                  # 12 feature pages (React components)
│   │   │   ├── FarmerAssistant.tsx  # AI chat with voice + web search
│   │   │   ├── CropCare.tsx         # Disease detection + hotspot map
│   │   │   ├── Community.tsx        # Q&A forum + location map
│   │   │   ├── EquipmentRental.tsx  # Rental + owner dashboard + payments
│   │   │   └── ...                  # 8 more feature pages
│   │   ├── components/             # Reusable UI components
│   │   │   ├── LanguageSwitcher.tsx  # 11-language selector
│   │   │   ├── Sidebar.tsx          # Navigation with i18n
│   │   │   └── ...
│   │   ├── context/                # Auth context (React Context API)
│   │   ├── services/api.ts         # API service layer
│   │   └── i18n/
│   │       ├── index.ts            # i18n config + font switching
│   │       └── locales/            # 11 language JSON files (291 keys each)
│   └── index.html                  # Entry point + Indic font imports
├── data/                           # JSON data storage (users, equipment, etc.)
├── Disease prediction Model/       # Xception CNN model files
├── Water Footprint Model/          # Random Forest model (joblib)
└── Farmer AI Assistant/            # RAG documents for ChromaDB
```
